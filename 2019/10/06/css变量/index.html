css变量

Posted on 2019-10-06
一.什么css变量

简单来说，类似于通常的变量，可以进行赋值和读取。
CSS 变量当前有两种形式：

  • 变量，就是拥有合法标识符和合法的值。可以被使用在任意的地方。可以使用var()函数使用变量。例如：var(–example)会返回–example所对应的值
  • 自定义属性。这些属性使用–where的特殊格式作为名字。例如–example: 20px;即使一个css声明语句。意思是将20px赋值给–example变量。
二.为什么要了解css变量

关于为什么在 CSS 中使用变量有很多理由，其中最主要的一点就是它减少了样式代码的重复
这样一来你的代码不仅仅是变得更加可读，而且也为你将来想要做出改变提供了更多的灵活性。
除此之外 CSS 变量还有很多亮点：

  • 它不需要经过任何转译步骤，因为它是浏览器原生支持的。所以你不需要像使用 SASS 和 LESS 那样，你不需要任何设置就能直接书写这样的代码。
  • 它是 DOM 的一部分，JavaScript可以直接访问

三.变量的声明

声明变量的时候，变量名前面要加两根连词线(–）。

body {
  --a: #7F583F;
  --b: #F7EFD2;
}
.content {
  --c: #F7EFD2;
}
:root {
  --main-color: #06c;
}

1.上面代码中，body选择器里面声明了两个变量：–a和–b,他们其实与常规的color，height等正式属性没什么不同，只是他们没有默认值，而这个默认值由你自己给出，所以 css 变量叫自定义属性。
2.body，content，root表示变量的作用域，root中放置全局变量，保证任何选择器都可以读取他们。

四.变量的读取

通过 var() 获取，例如

a {
  color: var(--a);
  text-decoration-color: var(--b);
}
五.JavaScript如何访问css变量
// 设置变量
document.body.style.setProperty('--primary', '#7F583F');
// 读取变量
document.body.style.getPropertyValue('--primary').trim();
// 删除变量
document.body.style.removeProperty('--primary');

通过JavaScript对css变量的操作，就可以实现用户自定义网页样式，或者一些十分炫酷的渐变效果
下面是一个利用 css 变量实现悬浮动画效果
css部分

.button {
position: relative;
appearance: none;
background: #f72359;
padding: 1em 2em;
border: none;
color: white;
font-size: 1.2em;
cursor: pointer;
outline: none;
overflow: hidden;
border-radius: 100px;
}

span {
    position: relative;
}

.button::before {
    --size: 0;  
    content: '';
    position: absolute;
    left: var(--x);
    top: var(--y);
    width: var(--size);
    height: var(--size);
    background: radial-gradient(circle closest-side, #4405f7, transparent);
    transform: translate(-50%, -50%);
    transition: width .2s ease, height .2s ease;
}

.button:hover::before {
    --size: 400px;
}

1.radial-gradient：由原点(渐变中心)辐射开的颜色渐变

js部分：得到鼠标的位置

document.querySelector('.button').onmousemove = (e) => {
    const x = e.pageX - e.target.offsetLeft
    const y = e.pageY - e.target.offsetTop

    e.target.style.setProperty('--x', `${ x }px`)
    e.target.style.setProperty('--y', `${ y }px`)
}

1.pageX、pageY：:对于整个页面来说，包括了被卷去的body部分的长度
2.offsetLeft、offsetTop：相对于带有定位的父盒子的左边和上面的坐标
